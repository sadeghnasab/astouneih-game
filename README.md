# Astouneih Game (بازی استونیه) 🎮

Astouneih is an open-source educational game developed with **Flutter** and **Dart**, dedicated to preserving, promoting, and teaching the endangered **Tati language** to the community. 

Through gamification and interactive learning, this project aims to engage younger generations and provide a scalable framework for minority language preservation. The application is fully functional and successfully published on the Myket platform for regional users.

## 🌟 Key Features
* **Interactive Gameplay:** Engaging levels designed to teach vocabulary and pronunciation.
* **Cultural Preservation:** A dedicated focus on the Tati language and regional heritage.
* **Cross-Platform:** Built using a single Flutter codebase for high performance.
* **Open Source AI Integration:** Preparing to integrate advanced LLMs (like Claude) for dynamic linguistic data generation and intelligent tutoring.
